# My discord bot

The idea of this project is to create a Discord bot for Among Us that sends the audio of all living players to the channel of the dead players.

## Requirements

Node version manager : 0.37.2
Node : 15.3.0 (nvm install node, nvm use node)
Node package manager : 7.0.14 (automatically installed along with Node)

Packages: discord.js, eslint
